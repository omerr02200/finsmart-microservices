# Finsmart Microservices
Bu proje, mikroservis mimarisiyle geliştirilmiş bir ön muhasebe/finans yönetim sistemidir.

## Teknolojiler
- Java 21 / Spring Boot 3.5.13
- **Spring Security & JWT** (Refresh Token destekli)
- **PostgreSQL** (Ana veritabanı)
- **Redis** (Token Blacklist & Caching)
- **Flyway** (Veritabanı Versiyonlama)
- **Docker & Docker Compose**
